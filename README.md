# CS303_Assignment1
Steps to Run
-------------
Ensure A1input.txt exists in the same folder as the program.
Run the program.
Display the menu with choices 1–6.
Repeat until the user chooses 6 (Exit):
Get the user’s menu choice.
Perform the corresponding action:
    1: Find an integer.
    2: Modify an integer at a specified index.
    3: Add a new integer to the end of the array.
    4: Remove an integer by index.
    5: Display the current list of integers.
    6: Exit the program.
    Validate all inputs (numbers and indices) and handle errors.
Close the program.
